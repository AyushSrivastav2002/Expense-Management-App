Expense Management Tracker Application developed totally in React and beautiful UI using tailwind CSS

User can add the Amount, Description about the amount, and categories (Income/expense) and can simply add them.

The added expenses can be shown on the website in the form of Table .

The added expenses will stored in the Local Storage.

User can also delete the added espense.
